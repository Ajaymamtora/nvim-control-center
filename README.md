# nvim-control-center

A customizable control center for Neovim to manage settings and options.

## Highlights

The following highlight groups are defined by default and linked to standard Neovim groups. You can override them in your configuration or theme.

| Group | Default Link | Description |
|---|---|---|
| `NvimControlCenterPanel` | `NormalFloat` | The main floating window background and text. |
| `NvimControlCenterBorder` | `FloatBorder` | The border of the floating window. |
| `NvimControlCenterTitle` | `FloatTitle` | The title of the window. |
| `NvimControlCenterSeparator` | `FloatBorder` | Separator lines within the UI. |
| `NvimControlCenterTabActive` | `CursorLine` | The currently active tab. |
| `NvimControlCenterTabInactive` | `NormalFloat` | Inactive tabs. |
| `NvimControlCenterTabIconActive` | `Special` | Icon for the active tab. |
| `NvimControlCenterTabIconInactive` | `Comment` | Icon for inactive tabs. |
| `NvimControlCenterStatusLine` | `StatusLine` | The footer/status line. |
| `NvimControlCenterLineActive` | `CursorLine` | The currently selected setting line. |
| `NvimControlCenterLineInactive` | `NormalFloat` | Unselected setting lines. |
| `NvimControlCenterIconActive` | `Special` | Icon for the selected setting. |
| `NvimControlCenterIconInactive` | `Comment` | Icon for unselected settings. |
| `NvimControlCenterSpacer` | `Title` | Section headers/spacers text. |
| `NvimControlCenterSpacerIcon` | `Constant` | Icon next to spacers. |
| `NvimControlCenterSpacerLine` | `Comment` | Decorator lines for spacers. |
